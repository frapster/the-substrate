the flagship · all four guarantees

One governed
decision, end to end

One intent, five deterministic checkpoints wrapped around a single model step. A good decision commits, is audited, and is reversible. A bad one is refused at the exact stage that owns the broken check.

the loop

Bounded, evidence-backed, validated, audited, reversible

Run the well-formed intent and watch all five stages pass, the audit chain verify, and a restore reproduce the prior state exactly. Then break one stage at a time and watch the refusal fire there, with nothing partial committing. The substrate is a pipeline of checkpoints, one model step in the middle.

the pipelinefive stages, run in your browser

Each "break" preset ships an intent broken in exactly one way. The refusal names the stage that owns that check: over-scope and unregistered are caught by bounded, a missing source by evidence, a malformed proposal by validated. A refused intent never reaches the ledger or the versioned store.

Don't take this at face value

This page is a JavaScript reimplementation of demos/governed-decision/pipeline.py, with the same stages, the same refusal conditions, and the same reason strings. It is a convenience. The proof is the Python you run yourself:

git clone https://github.com/frapster/the-substrate && cd the-substrate
python demos/governed-decision/governed_decision_demo.py   # one intent, five stages
python demos/governed-decision/test_pipeline.py            # each stage refuses its own failure
python demos/run_tests.py                                  # all 8 demo suites at once

Read the source: pipeline.py · README. Each stage has a fuller standalone proof: bounded, evidence, validator, audit, reversible.

Prove it yourself · extend the tests

The tests in demos/governed-decision/test_pipeline.py assert each stage refuses its own bad input and that a refusal leaves no trace. Add your own broken intent and confirm it is caught at the stage you expect:

def test_my_broken_intent(self):
    s = _substrate()
    intent = _good()
    intent.scope = {"usd": 100000}      # over the grant_refund hard cap
    d = s.decide(intent)
    self.assertFalse(d.committed)
    self.assertEqual(d.refusal_stage, "bounded")   # is it caught where you expect?
    self.assertEqual(len(s.ledger.rows), 0)         # nothing partial committed

Run python demos/governed-decision/test_pipeline.py. If you can craft a broken intent that commits anyway, or that leaks a partial write, you have found a real gap. You hold the falsifier.
